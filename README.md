# 💰 Skypro Wallet — Учет личных расходов

Приложение для ведения личных финансов: добавление, удаление, анализ расходов по категориям и периодам.  
**Полностью интегрировано с реальным API** (авторизация, хранение расходов).

---

## 🚀 Функциональность

- ✅ Регистрация и вход через API (`/api/user` и `/api/user/login`)
- ✅ Хранение расходов на сервере (Bearer token)
- ✅ Создание, просмотр, удаление расходов
- ✅ Категории: Еда, Транспорт, Жилье, Развлечения, Образование, Другое
- ✅ Аналитика расходов за выбранный период (календарь, диапазон дат)
- ✅ Визуализация: столбчатая диаграмма расходов по категориям
- ✅ Интерактивные формы с валидацией, понятные сообщения об ошибках

---

## 🧰 Технологии

- **React 18** (функциональные компоненты, хуки)
- **React Router DOM** — маршрутизация
- **Styled Components + CSS Modules** — стилизация
- **React Icons** — иконки
- **Fetch API** — HTTP-запросы
- **date-fns** — работа с датами

## 📦 Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/ваш-username/skypro-wallet.git

# Перейти в папку проекта
cd skypro-wallet

# Установить зависимости

# Запустить в режиме разработки

# Открыть в браузере http://localhost:5173
